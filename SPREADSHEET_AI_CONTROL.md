# 🎨 Spreadsheet AI Control - Full Feature Guide

## Overview
Your AI now has **full control** over your Google Spreadsheet! You can ask it to format, style, and modify the spreadsheet in natural language.

## 🚀 How to Use

### Method 1: Format Sheet Button (Header)
Click the purple **"Format Sheet"** button in the header for quick formatting options:
- 🦓 Alternate Row Colors (zebra striping)
- 📏 Auto-Resize Columns
- 🔍 Add Filter
- ❄️ Freeze First 2 Columns
- 🔤 Sort by Company Name

### Method 2: Chat with AI
Just ask in natural language! Examples:

**Formatting:**
- "Make every other row lighter"
- "Make every other row a lighter color"
- "Add zebra striping to the spreadsheet"

**Columns:**
- "Resize all columns to fit the content"
- "Auto-size the columns"
- "Freeze the first 2 columns"
- "Freeze column 1 and 2"

**Filters & Sorting:**
- "Add a filter to the spreadsheet"
- "Enable filtering"
- "Sort by company name"
- "Sort by sponsorship score descending"

## ✨ Available Actions

### 1. Alternate Row Colors (Zebra Striping)
Makes every other row light gray for easier reading.
```
Ask: "make every other row lighter"
```

### 2. Auto-Resize Columns
Automatically adjusts column widths to fit content.
```
Ask: "resize all columns"
```

### 3. Add Filter
Adds dropdown filters to the header row.
```
Ask: "add filter"
```

### 4. Freeze Columns
Keeps columns visible while scrolling horizontally.
```
Ask: "freeze first 2 columns"
Ask: "freeze column 1"
```

### 5. Sort Data
Sort by any column in ascending or descending order.
```
Ask: "sort by company name"
Ask: "sort by sponsorship score descending"
```

## 💡 Tips

1. **Be Specific:** The more specific your request, the better!
   - Good: "Make every other row light gray"
   - Better: "Add alternating row colors starting from row 2"

2. **Combine Requests:** You can ask for multiple things at once!
   - "Resize columns, add filters, and freeze the first 2 columns"

3. **Refresh After Formatting:** Some changes may require a page refresh to see in the UI.

## 🚨 Important Notes

- Changes are applied **immediately** to your live spreadsheet
- All formatting is **non-destructive** - your data is safe
- You can always manually undo changes in Google Sheets (Ctrl+Z)
- The spreadsheet must be shared with the service account

## 🆘 Troubleshooting

**"Permission denied" errors:**
- Make sure the spreadsheet is shared with: `sponsor-research@techbirmingham.iam.gserviceaccount.com`
- Give it "Editor" permissions

**Changes not showing:**
- Refresh your Google Sheets tab
- Check the terminal for error messages

**AI doesn't understand:**
- Try using the Format Sheet button for quick actions
- Use simpler language: "add filter" instead of "please enable filtering functionality"

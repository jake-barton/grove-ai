## 🆘 Getting Help

### Common Issues & Solutions

**"AI not responding"**
→ Start Ollama: `ollama serve`

**"Database error"**  
→ Check Supabase credentials in .env.local

**"Export failed"**
→ Verify Google Sheets service account access

**"No search results"**
→ Check Serper API key is valid
